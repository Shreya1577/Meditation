# Master Prompt: ZenFlow Meditation & Live Learning Platform

> **Task**: Build a premium, full-stack meditation platform named "ZenFlow" using the MERN stack (MongoDB Atlas, Express, Node.js, React + Vite).
>
> ### 1. Visual Identity & UI/UX
> *   **Theme**: Royal Blue & Cream (or Bluish Purple Glassmorphism).
> *   **Aesthetic**: High-end, "Zen," calm, and professional. Use smooth CSS transitions, subtle micro-animations, and a modern typography set.
> *   **Layout**: Fully responsive with a focus on a "distraction-free" student experience.
>
> ### 2. Core Feature Specifications
>
> #### A. Landing Page & Overview
> *   **Header**: Elegant navigation (Home, Courses, Dashboard, Login).
> *   **Hero Section**: A high-impact background image/gradient with a centered meditation quotation.
> *   **Call to Action (CTA)**: "Start Learning" and "Explore Courses".
>
> #### B. Authentication System
> *   **Signup/Login**: Clean, secure JWT-based authentication for Students and an Admin role.
>
> #### C. Course & Payment Flow (Manual Verification)
> *   **Course Catalog**: Display courses with Monthly and Yearly subscription options.
> *   **"Buy This Course" Logic**: Clicking buy triggers a Modal/Page showing **Admin Contact Information** (Phone, Payment ID, Instructions). 
> *   **Purchase Flow**: Student contacts admin -> Admin manually approves student.
>
> #### D. Student Dashboard
> *   **Progress Tracker**: Visual bars showing daily progress.
> *   **Subscription Status**: A countdown timer showing "Days Remaining".
> *   **Live Class Section**: A card that displays a **Zoom or Google Meet link** (Admin-controlled).
> *   **Schedule**: Morning and Evening session times.
> *   **Personalization**: One randomly generated "Daily Affirmation" and course suggestions.
>
> #### E. Admin Panel
> *   **User Management**: List of students and their courses.
> *   **Subscription Toggle**: Manually activate courses for students.
> *   **Live Link Broadcaster**: Input field to update the Zoom/Meet link for all students.
>
> #### F. Contact & Footer
> *   **Admin Profile**: Phone, Email, LinkedIn, and Instagram.
